# Online-Books-Store.-PROJECT

## Project Overview

This project demonstrates SQL-based data analysis on an Online Book Store dataset using PostgreSQL and pgAdmin 4.
The data was imported from three CSV files: Books, Customers, and Orders.
The objective was to perform business analysis, extract meaningful insights, and answer real-world business questions using SQL queries.


## Tools & Technologies

* PostgreSQL
* pgAdmin 4
* SQL
* CSV Datasets


## Dataset Information

The project uses three datasets:

### Books Table

Contains information about books such as:

* Book ID
* Title
* Author
* Genre
* Price
* Stock Quantity

### Customers Table

Contains customer details such as:

* Customer ID
* Name
* City
* Country

### Orders Table

Contains order information such as:

* Order ID
* Customer ID
* Book ID
* Order Date
* Quantity Purchased


## Project Objectives

* Analyze book sales performance
* Identify top-selling books
* Find high-value customers
* Calculate total revenue generated
* Analyze customer purchasing behavior
* Track inventory and stock levels


## Key SQL Concepts Used

* SELECT Statements
* WHERE Clauses
* ORDER BY
* GROUP BY
* Aggregate Functions (SUM, COUNT, AVG)
* JOIN Operations
* Subqueries
* Date Functions


## Business Insights Generated

* Top-selling books by quantity sold
* Most valuable customers based on purchases
* Total revenue generated from book sales
* Best-performing genres
* Monthly sales trends
* Customer order patterns
* Inventory status analysis
